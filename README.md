# GG Media Kit

Media kit site for **A Global Gathering for the Future of Child Welfare**. Static HTML/CSS/JS — no build step.

## Run locally

```
npx serve .
```

or just open `index.html` in a browser.

## Structure

- `index.html` — page markup and content
- `styles.css` — design system (colors, type, components) per brand guardrails
- `assets.js` — media asset manifest (logos, banners, graphics, video embeds)
- `script.js` — renders asset cards from the manifest, handles copy-to-clipboard actions
